## Project Overview

This project aims to predict whether a student will **pass** or **fail** using supervised machine learning techniques.

Using the **Student Performance Dataset**, we analyze academic, personal, family, and social factors to build classification models capable of predicting students' final outcomes. The project includes data preprocessing, exploratory data analysis, model training, evaluation, and comparison.

The target variable (`passed`) is derived from the final grade (`G3`):

- **Pass:** G3 ≥ 10
- **Fail:** G3 < 10
